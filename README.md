## 🚀 **CARA INSTALL (UNTUK YANG BARU)**

### **Langkah 1: Install Termux**
```
📱 Buka Play Store/F-Droid
🔍 Cari "Termux"
📲 Klik Install
▶️ Buka apps Termux
```

### **Langkah 2: Setup Awal**
```bash
pkg update && pkg upgrade -y
pkg install git -y
```

### **Langkah 3: Download Script**
```bash
git clone https://github.com/username/termux-all-package
cd termux-all-package
```

### **Langkah 4: Jalan Script**
```bash
chmod +x install-all.sh
./install-all.sh
# atau
bash install-all.sh
```

### **Langkah 5: Tunggu Siap**
> ⏳ Proses install akan ambil masa **30 minit - 2 jam** bergantung pada internet korang.
